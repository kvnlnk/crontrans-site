Precision features

01

Bidirectional

Translate cron → English for reading, or English → cron for writing. Never second-guess a schedule again.

02

Zero dependencies

Written in pure Python with no external requirements. Install once, run anywhere cron lives.

03

All formats supported

Standard 5-field cron, @-shorthands, ranges, step values, lists, and special strings — everything cron can express.

04

CI-ready

Pipe cron expressions through crontrans in scripts, CI pipelines, or monitoring alerts. Human-readable schedules everywhere.

Common patterns

0 2 * * *
→
every day at 2:00am
*/5 9-17 * * 1-5
→
every 5 minutes, Monday through Friday, 9am to 5pm
0 0 1 1 *
→
January 1st at midnight

Install

One command, zero fuss. Requires Python 3.8+.

pip install crontrans

Then: crontrans "*/15 9-17 * * 1-5"
